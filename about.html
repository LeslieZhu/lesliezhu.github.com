说明

此博客 YiLife 主要写一些心情随笔和技术文章，博主是一位程序员，日常主要使用 Python 和 C++ 进行软件开发.

此博客基于OrgNote生成，它基于 freemind 主题定制，将 org-mode 文件在 Emacs 转换为网页文件基础上，再进行简单处理以适应此主题。

这是一个为自己定制的博客工具，以满足自己的需求为主，功能相对简单，如果有兴趣使用可以直接安装: 

$ pip install orgnote

支持 Python2 和 Python3, 如果遇到什么问题，可以提交issue以帮助改进。
