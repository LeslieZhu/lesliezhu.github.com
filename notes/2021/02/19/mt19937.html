mt19937: C++11伪随机数产生器

1 前言

  • https://www.jianshu.com/p/6d9a7de995bb
  • https://www.cplusplus.com/reference/random/mt19937/

std::mt19937是伪随机数产生器，用于产生高性能的随机数。 C++11引入。

返回值为unsigned int。

名称由来: 

mt是因为这个伪随机数产生器基于Mersenne Twister算法。

19937是因为产生随的机数的周期长，可达到2^19937-1。

std::mt19937 接收一个unsigned int数作为种子。所以可以如下定义： 

std::mt19937 mt_rand(std::random_device{}());
std::mt19937 mt_rand(time(0));
std::mt19937 mt_rand(std::chrono::system_clock::now().time_since_epoch().count());

2 举例:随机2个字符

int nChars = 2;
mt19937 generator{random_device{}()};
uniform_int_distribution<int> dist{'a', 'z'};
auto randchar = [&] () -> char { return dist(generator); };

auto InStrPair = sequence<stringIntPair>::from_function(nRcds, [&](size_t i){
    charSeq rstr(nChars);
    generate_n(rstr.begin(), nChars, randchar);
    return stringIntPair(rstr, i); });
writeSequenceToFile(InStrPair, "junk.str");

3 举例: 随机5个数

#include <iostream>
#include <random>

using namespace std;

int main()
{
    std::mt19937 rng(std::random_device{}());
    for (int i = 0; i < 5; i++) {
        cout << rng() << endl;
    }

    return 0;
}

4 举例：特定分布

分布有很多种，如均匀分布、正态分布等，可参考(https://zh.cppreference.com/w/cpp/numeric/random)

产生特定分布的随机数，需要使用分布函数

正态分布: 

#include <iostream>
#include <random>

using namespace std;

int main()
{
    std::mt19937 rng(std::random_device{}());
    std::normal_distribution<double> nd(5, 2);
    for (int i = 0; i < 5; i++) {
        cout << nd(rng) << endl;
    }

    return 0;
}

执行： 

8.80156
5.78055
4.99761
7.6995
1.21933
