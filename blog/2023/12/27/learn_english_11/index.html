    什么是MBTI性格测试？

    前言¶

    注：这是一篇翻译，英语原文标题 Personality Test: The Myers-Briggs Type Indicator (MBTI)，作者是Mystic Flores。

    MBTI性格测试的全称是The Myers-Briggs Type Indicator Personality Test。

    如果你想了解它，那么这篇文章包含了所有你需要的信息。

    MBTI是一个非常受欢迎的性格测试工具，由Isabel Briggs Myers和她的妈妈Katharine Cook Briggs在20世纪共同创立。

    起源¶

    MBTI是从Carl Jung的工作基础上发展而来的，他认为有四种基础心理功能：

    • 外向型（Extraversion, E）和内向型（Introversion, I）：人们获取和扩展能量的方式。

      外向型是从周围人身上获取能量并积极参与活动。内向型是靠自己本身获取能量，喜欢反思自己的想法和感受。（按：关注力，是喜欢关注他人并获得外驱力，还是更多的关注自己内心产生内驱力。）

    • 感觉型（Sensing, S）和直觉型（Intuition, N）：人们聚合信息的方式。

      感觉型偏向于具体的事实和细节。直觉型偏向于模式、可能性和抽象概念。（按：认知方式，是喜欢具像化的现实中实实在在的东西，还是喜欢抽象化的概念性东西，富有想象力。）

    • 理智型（Thinking, T）和情感型（Feeling, F）：人们做决定的思维方式。

      理智型偏向于根据逻辑和根由来做出决定。情感型偏向于价值观念和情绪来做出决定。（按：决策方式，是理性还是感性。）

    • 判断型（Judging, J）和感知型（Percteiving, P）：人们如何安排自己的生活。

      判断型偏向于有条理、有组织、有结构的生活。感知型偏向于更加灵活和自主的生活。（按：行为方式，是喜欢先列计划清单再有条不紊的去执行，还是率性而为。）

    排列组合¶

    MBTI将这4种基础功能排列组合成16种性格类型，每一种都有它本身独有的优点和缺点。基于这4个进行二分法，MBTI归纳出16种不同的个性类型：

    • ISTJ — The Inspector : 警长
    • ISFJ — The Defender : 战士
    • INFJ — The Advocate : 律师
    • INTJ — The Architect : 建筑设计师
    • ISTP — The Craftsman : 工匠
    • ISFP — The Artist: 艺术家
    • INFP — The Mediator : 调解人
    • INTP — The Logician : 逻辑学家
    • ESTP — The Entrepreneur : 企业家
    • ESFP — The Entertainer : 表演者
    • ENFP — The Inspirer : 启发者
    • ENTP — The Visionary : 梦想家
    • ESTJ — The Executive : 执行者
    • ESFJ — The Consul : 领事、外交家
    • ENFJ — The Teacher: 教师、教练、主人公
    • ENTJ — The Commander : 指挥官

    这个测试是设计用来帮助人们了解自己的个性偏好，人们可以使用这些偏好来改善自己的沟通、关系和职业。

    MBTI已经被广泛用于各种设置中，包括商业、教育和个人发展。当然这项测试也因为缺乏科学依据和过度简化人类的个性定义而受到批评。尽管有这些批评，但MBTI依旧是受欢迎的自我评估和个人发展工具。

    历史¶

    这是MBTI发展历史的大概时间线：

    • 1921年: Carl Jung发表了他的书籍《心理类型》，建立了MBTI的基础。
    • 1921年后：Isabel Briggs Myers开始学习Jung的理论并发展出她自己的个性评估工具。
    • 1942年: Myers和她的妈妈Katharine Cook Briggs发表了MBTI的第一个版本。
    • 1950年代和1960年代：MBTI在美国广受欢迎，特别是在商界。
    • 1970年代和1980年代:MBTI在欧洲和其它一些地方越来越受欢迎。
    • 1990年代和2000年后：MBTI的使用更加广泛，对它缺乏科学依据的批评声音也与日俱增。
    • 2010年后: MBTI依旧是很受欢迎自我评估和个性测试工具，但它越来越多地被用于和其它评估工具结合使用。

    有效性¶

    一系列问题导致MBTI在评估个性时不那么可靠和准确。

    • 缺乏科学依据：MBTI是基于Carl Jung的理论，但并没有进行严格的测试或得到科学研究的支持。没有证据表明MBTI对于衡量个性有意义，并且它的结果随着时间推移或交叉评估时不总是一致的。
    • 二分法思维：MBTI将人们分成16种性格类型，它是基于4种二分法而来的，外向型或内向型、感觉型或直觉型、理智型或情感型、判断型或感知型。这种二分法思维过于简单粗暴，不适应于人类复杂的性格。
    • 巴纳姆效应：MBTI总是用模糊和笼统的方式来描述性格，这产生一种广为人知的巴纳姆效应。人们倾向于相信模糊的性格描述和自己相匹配，即使这些描述适用于任何人。
    • 缺乏预测能力：MBTI无法很好的预示行为或表现。没有证据表明拥有相同MBTI类型的人会在发展职业和人际关系上能取得相似的成就。
    • 职场上的误用：MBTI常被用于职场帮助决策，比如招聘、晋升和团队建设。然而，并没有证据表明MBTI在这些方面有效。事实上，用这种方式使用MBTI会导致歧视和不公平对待。
    • 对于性格的理解有限：MBTI只关注一小部分性格特征，而不综合考虑其它重要的因素，比如认知能力、情商和动机。
    • 商业化：MBTI是一个有版权的方法，并且仅限于认证从业者使用。这导致较少被研究者涉及，从而导致独立测试研究的数量偏少。

    总之，MBTI是一个有缺陷的不那么可靠的性格评估工具。它缺乏科学依据，过度简化性格，也不能很好的预示行为。它不应该用于个人重要的决策，而且要停止在职场的使用。

    标准石油公司：使用MBTI的先驱¶

    标准石油公司是历史上最大、最成功的企业之一，也是最早一批使用MBTI评估工具的。这家企业从1950年代开始将MBTI用于招聘和晋升，并且迅速变成企业文化中不可或缺的组成部分。

    标准石油公司发现MBTI是一个物有所值的工具，它可以帮助他们为岗位辨认和雇佣最好的应聘者。这家企业同时也使用MBTI来帮助雇员发展自己的技能和发挥自己的潜能。

    现在，依旧有很多企业使用MBTI，它被认为是一个有效的工具，用于理解和管理雇员的行为。然而，有一个很重要的地方需要注意，那就是MBTI不是一个完美的工具，它不应该作为做出招聘、晋升和培训决定的唯一参考。

    补充：后缀¶

    在现有四类心理基础上，发展出第五种性格心理，即：

    • A（Assertive）：坚决的、自信的，对自己的观点深信不疑
    • T（Turbulent）：动摇的、动荡的，容易动摇自己的观点或者怀疑自我

    比如主人公性格 ENFJ-A / ENFJ-T和调停者性格 INFP-A / INFP-T。
